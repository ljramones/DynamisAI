# dynamis-ai-voice

Voice pipeline contracts for TTS orchestration, rendering backends, and expressive output control.

Part of the DynamisAI engine stack.

## Build

Requires JDK 25 and Maven 3.9.x.

```bash
mvn clean install
```

## Dependencies

- `org.dynamisai:dynamis-ai-core:1.0.0-SNAPSHOT`
- `org.dynamisai:dynamis-ai-cognition:1.0.0-SNAPSHOT`
- `ai.djl:api:0.30.0`
- `ai.djl.onnxruntime:onnxruntime-engine:0.30.0`

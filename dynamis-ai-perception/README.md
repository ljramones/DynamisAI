# dynamis-ai-perception

Perception interfaces for simulated senses, saliency filtering, and world-to-AI observation shaping.

Part of the DynamisAI engine stack.

## Build

Requires JDK 25 and Maven 3.9.x.

```bash
mvn clean install
```

## Dependencies

- `org.dynamisai:dynamis-ai-core:1.0.0-SNAPSHOT`
- `ai.djl:api:0.30.0`

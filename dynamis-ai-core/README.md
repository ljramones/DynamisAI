# dynamis-ai-core

WorldStateStore, BudgetGovernor, AI frame scheduling, and core data contracts
for the DynamisAI engine.

Part of the DynamisAI engine stack.

## Build

Requires JDK 25 and Maven 3.9.x.

```bash
mvn clean install
```

## Dependencies

None. This is the zero-dependency leaf of the DynamisAI graph.

# dynamis-ai-social

Social system contracts for relationships, reputation, and group-level behavior drivers.

Part of the DynamisAI engine stack.

## Build

Requires JDK 25 and Maven 3.9.x.

```bash
mvn clean install
```

## Dependencies

- `org.dynamisai:dynamis-ai-core:1.0.0-SNAPSHOT`
- `org.dynamisai:dynamis-ai-cognition:1.0.0-SNAPSHOT`
- `org.dynamisai:dynamis-ai-memory:1.0.0-SNAPSHOT`

# dynamis-ai-navigation

Navigation contracts for pathfinding, steering, and traversal planning based on perception-derived inputs.

Part of the DynamisAI engine stack.

## Build

Requires JDK 25 and Maven 3.9.x.

```bash
mvn clean install
```

## Dependencies

- `org.dynamisai:dynamis-ai-core:1.0.0-SNAPSHOT`
- `org.dynamisai:dynamis-ai-perception:1.0.0-SNAPSHOT`

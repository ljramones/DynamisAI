# dynamis-ai-cognition

Async cognition service contracts and inference integration boundary for DynamisAI.

Part of the DynamisAI engine stack.

## Build

Requires JDK 25 and Maven 3.9.x.

```bash
mvn clean install
```

## Dependencies

- `org.dynamisai:dynamis-ai-core:1.0.0-SNAPSHOT`
- `com.github.tjake:jlama-core:0.8.3`
- `dev.langchain4j:langchain4j:0.36.0`
